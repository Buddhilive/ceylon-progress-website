# Framework Integration

**Ceylon Progress** can be used in any popular frontend frameworks available. **Ceylon Progress** can be installed via any Node.js package manager;

```shell
npm i ceylon-progress

or

yarn add ceylon-progress
```

> For more details on how to install **Ceylon Progress** in a specific framework, please refer the official [Stencil.js Framework Integration Guide](https://stenciljs.com/docs/overview).
