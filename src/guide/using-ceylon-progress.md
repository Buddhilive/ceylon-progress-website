# Ceylon Progress Options

There are variety of animations you can select. Parse the name of the animation to `animation-type` attribute of the component to change the default animation type. If you enter the animation name wrong, it will automatically switch to default animation. _Note: Names are case sensitive_

## Pulse

<div class="html-preview-block">
    <ceylon-dots animation-type="pulse"></ceylon-dots>
</div>

```html
<ceylon-dots animation-type="pulse"></ceylon-dots>
```
